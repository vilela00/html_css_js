Operadores JavaScript

Operadores lógicos e relacionais

== ou ===

O primeiro compara dois valores e retorna true ou false, o segundo além de comparar os valores, compara os tipos das variáveis, e caso uma das condições não seja atendida (valor ou tipo) retorna como false.

!= e !==

Diferente somente valores, ou valores e tipos.

<, >, <= e >=

Menor, maior, menor igual e maior igual.

&&

And ou E - utilizando também o sinal de != diferente.

||

Or ou OU - neste exemplo foi utilizado para generalizar a entrada de texto, com minúsculas, maiúsculas e mistas.

!

Not ou NÃO - este é usado para inverter o valor. Caso uma condição seja true, ele transforme em false, e vice-versa.

No exemplo acima perceba que o ! foi utilizado para transformar uma condição falsa, em verdadeira. Foi criada uma nova variável eVazio com o valor do nome sendo vazio. No if foi usado o ! para inverter a condição, que no caso, foi definida como TRUE para quando não houver nenhuma entrada. Neste caso, se não houver nenhuma entrada, aí sim ele será lido como false e vai ler o * else *. Se não houvesse o !, ele entederia que caso houvesse uma entrada qualquer, ele leria o * else *, e caso não houvesse nenhuma entrada, ele leria o * if * como true.
