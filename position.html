Position CSS

Define uma posição de um elemento em realção a outro (pai)

Static

Atribuído por padrão a todos os elementos que não possuem outro valor de position definido. Dessa forma nenhum elemento transpassa o outro. Eles são dispostos lado a lado ou empilhados sem nenhum espaçamento entre eles a não ser que seja definido.

Relative

O position relative é usado com a definição de uma posição usando propriedades de indicação para dizer onde o elemento será posicionado.

Ao utilizar o position relative atente-se à responsividade da página pois os elementos irão se sobrepor ao ter variação de viewport, diferente do static, que empurra o elemento para o lado e para baixo.

o atributo de position * absolute * precisa ser usado juntamente com o position *relative * em seu elemento pai.

Absolute

O position absolute é usado com a definição de uma posição usando propriedades de indicação para dizer onde o elemento será posicionado.

Ao utilizar o position relative atente-se à responsividade da página pois os elementos irão se sobrepor ao ter variação de viewport, diferente do static, que empurra o elemento para o lado e para baixo.

Perceba que o position absolute não acompanha a responsividade, pois quando é alterado o viewport, o elemento absolute permance travado na página.

Fixed

O position fixed faz com que o elemento fique preso ao viewport, a não ser que o um elemento pai tenha uma propriedade de transform, perspective ou filter definido como diferente de none. Neste caso, o pai se comporta como o container que o contém.

Sticky

O position sticky posiciona o elemento de acordo com o fluxo normal do documento e, em seguida, deslocado em relação ao pai de rolagem mais próximo.
