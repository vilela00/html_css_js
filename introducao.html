Introdução Front End HTML, CSS e JS

Primeiros passos no Dev Front End

Primeiramente será necessário realizar a instalação e configuração dos Softwares e Ferramentas básicas para o uso das linguagens.

O HTML é uma linguagem de diagramação utilizada para gerenciar tags de forma a incluir blocos por meio de códigos. As melhores práticas compreendem em sempre utilizar as tags semânticas header, nav, section, aside, article, footer, para melhor entendimento dos navegadores. Tente manter seu código limpo e organizado para que seja de fácil compreensão e tenha uma boa performance quando houverem tarefas complexas a serem executadas.

O CSS é uma linguagem de estilização e se torna uma poderosa ferramenta de personalização de design de um site ou app. Pode ser aplicado diretamente na tag HTML na mesma página onde está sendo desenvolvido o código, ou pode ser importado via link:css no head do documento, e por meio de classes ou id's que são utilizadas nas tags HTML. Essa é a melhor prática recomendada, com 1 ou vários arquivos .css para estilização do projeto.

O JavaScript é uma linguagem de programação que por meio de arrays, strings, vars, realiza diversas ações para o dinamismo do projeto.
