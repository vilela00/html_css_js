Variáveis JS

const, let e var

Variáveis no JavaScript são como caixas onde você guarda informações que quer usar depois, como números, textos ou objetos. Imagine que você está criando um jogo: você precisa armazenar a pontuação do jogador, o nome dele, e outras informações.

Diferenças entre as variáveis JS

Existem três tipos principais de variáveis no JavaScript: var, let e const. Var é a forma antiga e pode ser confusa porque tem escopo de função, o que significa que ela pode ser acessada de qualquer lugar dentro da função onde foi declarada. Let e const são mais modernos e têm escopo de bloco, o que significa que só podem ser acessados dentro do bloco onde foram declarados. A principal diferença entre let e const é que const não pode ser reatribuída depois de definida.

Quando usar cada variável

Use let quando você precisar de uma variável que pode mudar de valor ao longo do tempo. Use const para valores que não devem mudar, como configurações ou valores fixos. Evite usar var a menos que você esteja lidando com código legado que precisa manter.

Exemplos de Boas Práticas com as variáveis JS

Prefira sempre usar const a menos que você saiba que o valor precisará mudar, nesse caso, use let. Isso ajuda a prevenir bugs, porque fica claro que aquele valor não deve mudar. Organize seu código para minimizar o escopo das variáveis, declarando-as o mais próximo possível do seu uso.

Exceções e regras

Use var apenas se você estiver mantendo ou trabalhando em código antigo. Lembre-se: const para valores constantes e let para valores mutáveis. Evite misturar tipos de variáveis no mesmo bloco de código para manter a clareza.
