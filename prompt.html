Prompt

Caixa de interação com o usuário - solicita entrada de dados

Utiliza-se com uma const a ser definida pelo usuário, por exemplo. Pode ser configurado um prompt que irá solicitar algum dado ao usuário.

Defina o tipo de valor que será retornado pelo usuário para posteriores execuções. Se você está coletando um número para ser usado em alguma operação matemática, defina a const como Number. Se você irá juntar os dados, mesmo que numéricos, sem realizar operações matemáticas, tenha certeza de que a const está definida corretamente.

Pode ser utilizado o parseFloat() para definir como número matemático, ou Number().
