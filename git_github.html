GIT e GitHub

GitHub

Realizar cadastro/login no github.com;

Criar um novo repositório em "+new" - dar um nome ao projeto (será usado como endereço web para publicar via GitHub pages e Vercel);

Criar uma nova pasta local e abrir no VS Code;

Criar os primeiros arquivos e salvar;

Iniciar o terminal no VS Code;

Sequência de comandos GIT:

  • git init;
  • git remote - copiar link gerado na criação do repositório;
  • git add . - adicionar modificações realizadas no container de envio;
  • git commit -m "descrição da versão" - nomeia a alteração realizada;
  • git push origin master (ou nome da branch utilizada) - envia as alterações para o repositório;
  • git pull origin master (ou branch) - solicita os dados do repositório;
  • git clone - clona um repositório via link;
  • git checkout -b exemplo.exemplo - -b cria uma branch, checkout leva para a branch;
  • git log - mostra as alterações no repositório.
