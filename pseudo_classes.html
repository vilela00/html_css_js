Pseudo-Classes :

Palavra chave adicionada a seletores que especifica um estado especial do elemento selecionado.

Veja um material completo CLICANDO AQUI.

:active

Pseudo-classe atribuída a links. Ela deve ser utilizada com base na regra LVHA - :link :visited :hover :active.

Exemplo: Link de Teste

:checked

Representa um elemento input type radio ou checkbox, ou option em um select (caixa de seleção dropdown), que está marcada ou alternando para um estado ligado. O usuário pode mudar seu estado clicando no elemento.

A utilização da personalização destes elementos depende de resetar o webkit importado do Bootstrap ou qualquer outro framework, utilizando o -webkit-appearance: none;

Pode-se utilizar também os inputs checkbox ou radio escondidos (display none) para incorporar valores booleanos. Verifique um exemplo NESTE LINK, onde foi usado o checkbox escondido no botão seletor de temas.

:default

Também atribuída aos inputs radio, checkbox e option, incluíndo também o elemento button. É usado para estilizar os inputs que estão checked por padrão. Aqueles que já vem selecionados, além do preencimento padrão.

:dir()

Usado para estilizar linhas de texto com base na direção da leitura. Requer sempre um parâmetro para funcionar ltr, ou rtl (left to right ou rigth to left).

:disabled

Define um estilo para um elemento desabilitado.

:empty

Representa qualquer elemento que não tenha filhos ou blocos de texto, incluíndo, espaços vazios.

:enabled

Representa qualquer elemento ativado. Um elemento é ativado se puder ser selecionado, clicado, digitado ou aceitar foco.

:first

Utiliza a regra @page e representa a primeira página de um documento.

:first-child, :last-child, :nth-last-child()

Representa o primeiro filho. Dentro de um container, tendo várias divs, ao utilizar o first-child vocês está selecionando apenas a primeira div. Da mesma forma que o last-child seleciona a última div. O nth-last-child seleciona com base em uma posição, contando a partir do último elemento.

:first-of-type, :last-of-type, :nth-last-of-type()

Representa o primeiro elemento do seu tipo entre os filhos de seu pai, assim como o last representa o último. O nth-last-of-type seleciona os elementos de acordo com o tipo, com base em uma posição, contando a partir do último elemento.

:fullscreen

Corresponde a todos os elementos que estiverem em modo de tela cheia.

:focus

Aplicado quando um elemento recebe foco. Clicar em inputs de texto de um formulário, por exemplo.

:hover

Corresponde quando o usuário passa o mouse por cima, mas não clica. Pode ser substituída pelas outras pseudo-classes :link, :visited e :active.

:indeterminate

Representa qualquer elemento de um formulário que tem seu estado como indeterminado.

:in-range, :out-of-range

Define uma estilização de acordo com um limite de valores definidos no html, dentro ou fora do intervalo.

:invalid

Representa qualquer elemento input ou form que seja inválido.

:lang()

Representa elementos baseando na linguagem.

:left, :right

Utilizado com a regra @page, representa todas as páginas à esquerda ou à direita de um documento.

:link

Permite que sejam selecionados os links dentro de um elemento. Para funcionar corretamente, ele deve ser utilizado antes do :hover, :active ou :visited.

:not()

Uma pseudo-classe de negação, usada para estilizar caso um elemento seja especificado fora da negação. Se utiizar o p:not(.uma_classe), significa que quando o p tiver essa classe, não receberá o css.

:nth-child()

Seleciona elementos com base nas suas posições em um grupo de elementos irmãos. Sempre utilizado com um valor entre os ()

Exemplos de seletores: tr:nth-child(odd) ou tr:nth-child(2n+1) - seleciona todas as linhas ímpares de uma tabela; tr:nth-child(even) ou tr:nth-child(2n) - seleciona todas as linhas pares de uma tabela; :nth-child(7), seleciona o sétimo elemento.

:nth-of-type()

Seleciona elementos com base nos seus tipos e nas suas posições em um grupo de elementos irmãos. Sempre utilizado com um valor entre os ()

:only-child

Representa um elemento sem nenhum elemento irmão.

:only-of-type

Representa qualquer elemento que não possui irmãos de um mesmo tipo.

:optional

Representa qualquer elemento input, select ou textarea que não contenha o atributo required.

:read-only

Seleciona elementos que não são editáveis pelo usuário.

:read-write

Representa um elemento que é editável pelo usuário.

:required

Representa qualquer elemento que seja definido como required.

:root

Representa o documento html raiz.

:scope

Pode representa o html ou qualquer bloco onde ele seja especificado.

:target

Representa o destino após um link ou botão ser clicado.

:valid

Representa qualquer input ou elemento do form cujo conteúdo foi validado com sucesso.

:visited

Representa links que já foram visitados.
