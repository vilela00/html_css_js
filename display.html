Display CSS

Flex

Utilizado para alterar a exibição padrão dos elementos. As divs por exemplo, são exibidas em blocos, e por padrão, um bloco ocupa toda a faixa horizontal do viewport. Ao criar várias divs, elas serão exibidas uma embaixo da outra. Pode-se utilizar o display:flex para que essas divs sejam exibidas uma ao lado da outra. Essa direção também pode ser alterada de acordo com as preferências, sendo elas row e collum. A utilização do display flex possibilita o uso de algumas propriedades que facilitam na organização automática dos elementos exibidos. Essas propriedades são: justify-content, align-items e gap, que só funcionam com o flex ativado.

Grid

O display Grid é utilizado com a definição de cada elemento com um nome para que seja definida sua posição de acordo com as áreas de cada posição.
