HTML Semântico

Tags específicas para identificar as seções da página

header - usada para o cabeçalho;

nav - usada para menu de navegação;

section - usada para seções;

article - usada para artigos;

aside - usada para seções laterais;

footer - rodapé.

Além das tags deve-se seguir um padrão de estrutura no arquivo HTML conforme imagem abaixo. O head é uma tag específica para receber link de importação, meta tags, scripts e título da página.
